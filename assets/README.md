# App icons

`evidence-ledger-icon.png` is the original generated icon source.

The smaller PNG files are exported for browser and installed-app use:

- `evidence-ledger-icon-32.png`: browser favicon
- `evidence-ledger-icon-64.png`: header brand mark
- `evidence-ledger-icon-180.png`: Apple touch icon
- `evidence-ledger-icon-192.png`: PWA manifest icon
- `evidence-ledger-icon-512.png`: PWA manifest icon

All paths are relative so they work both at the repository root and under a GitHub Pages project subdirectory.
